# Personal Portfolio Website

## Overview

A modern and responsive **Personal Portfolio Website** developed to showcase professional profile, technical skills, education, and contact information. The project focuses on delivering an engaging user experience through interactive animations, responsive layouts, and clean UI/UX design.

## Features

* Responsive and mobile-friendly design for seamless cross-device compatibility
* Interactive animations and smooth scrolling effects using **GSAP**
* Dedicated sections for **About, Skills, Education, and Contact**
* Clean and reusable code structure with modern frontend development practices

## Technologies Used

* **HTML5** – Structure and semantic layout
* **CSS3** – Styling and responsive design
* **JavaScript (ES6+)** – Interactive functionality
* **GSAP** – Advanced animations and scroll effects

## Key Highlights

* Designed and developed a visually appealing developer portfolio website
* Implemented smooth navigation and user-friendly interface
* Optimized responsiveness for desktop, tablet, and mobile devices
* Showcased frontend development skills with modern UI/UX concepts
